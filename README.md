Problemas de la Vida Usando Matemática
Fibonacci y Números Primos aplicados a problemas del mundo real
Este proyecto es una página web interactiva que aplica la serie de Fibonacci y los números primos para resolver tres problemas del mundo real:
•	Colmena de Abejas.- el árbol de ancestros de un zángano sigue la serie de Fibonacci
•	Seguridad RSA.- los números primos son la base de la criptografía moderna
•	Crecimiento de Conejos.- simulación de población con Fibonacci y detección de 
Tecnologías usadas
•	HTML — estructura de la página
•	CSS — diseño visual y diseño responsivo
•	JavaScript — lógica de Fibonacci, números primos y manejo de formularios
Algoritmos implementados
•	Serie de Fibonacci
•	Verificación de número primo
